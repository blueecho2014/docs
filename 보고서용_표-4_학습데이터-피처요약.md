# 표 4. 학습 데이터 구성 및 입력 피처 요약(xAI 기초 모델)

아래 표는 HWP에 **표로 붙여넣기** 용도로 정리한 요약본입니다.

| 구분 | 내용 |
|---|---|
| 원천 데이터(예) | IRT 데이터(`data/irt_1216_all.csv`) + 답안/학습로그(`data/toctoc_1216.csv`) |
| 결합 키 | `rcset_id`, `question_id` 기준 결합 |
| 타겟(현 구현) | `answer`(정답 여부, 이진 분류) |
| 전처리(핵심) | 불필요 컬럼 제거, 결측 처리, `member_id`/`question_id` 재매핑(충돌 방지) |
| 입력 피처 범주(예) | 식별/세션(`member_id`, `rcset_id`, `question_id`), 행동/로그(`duration`, `response_count`), 학습 맥락(`grade_cd`, `semstr_cd`, `lsn_cd`, `mode_id`, `conts_dtl_qitem_type_se_cd`), IRT(`Discrimination`, `Difficulty`) |
