# 연구개발과제 수행 결과 및 목표 달성 정도(초안)

> 본 문서는 **연구개발계획서 기반**으로 1차년도 수행 결과를 **증빙(부록) 중심**으로 정리하고, 각 기술개발의 **진척도(%)** 및 **2026년(2차년도) 개발 일정/계획**을 포함한 초안이다.

---

## 1. 1차년도 연구개발 목표(계획서 기반)
- **1차년도 목표**: 선행 연구개발(AI 모델, 플랫폼) 기반하여 xAI 플랫폼의 **설명 가능한 모델**과 **LLM 모델**을 개발하고, 이를 시각화할 **설명 인터페이스 시스템**을 구축
- **1차년도 세부과업(요약)**  
  - (1) xAI 기초 모델 개발(수준/취약 진단·예측 기반)  
  - (2) SHAP Value 기반 해석 강화 및 sLLM/Agent 기반 마련  
  - (3) LLM 기반 설명 인터페이스 시스템(Agent+RAG, SaaS 연계) 개발/구축  

---

## 2. 1차년도 수행 결과(증빙 중심) 및 진척도(%)
> 진척도는 **(완료된 개발·산출물) / (계획서 기준 요구 기능·검증까지 포함한 전체 범위)** 관점에서 산정하였다. 1차년도에 **프로토타입 UI/연동 및 공인시험성적서 기반 검증은 미수행**임.

### 2.1 (1) xAI 기초 모델 개발 — **진척도 80%**
- **계획서 기반 개발 내용**: 학습자 수준 및 취약 영역을 진단/예측할 수 있는 xAI 기초 모델 개발
- **수행 결과(무엇을 개발했는가)**  
  - IRT 데이터 및 답안/학습 로그 데이터를 결합하여 학습 데이터 구성 및 전처리 파이프라인 구현  
  - RandomForest 기반 예측 모델 학습/평가 파이프라인 구축  
  - SHAP 적용을 위한 입력 피처 구성 및 샘플 추출 로직 구현  
- **정량 확인(내부 파일럿)**  
  - Accuracy **0.9076**, AUC **0.6626** *(내부 실행 기반 파일럿 결과)*  
- **증빙(부록 참조)**  
  - 시스템/파이프라인 요약: **부록 B-1**  
  - 모델 정의/성능/재현 절차: **부록 B-3**  
- **미진행 및 보완(2차년도 이월)**  
  - 계획서 정량지표(수준진단/취약진단/해석 모델 정확도 등)와의 **지표 정의 정합화**, **평가셋 고정**  
  - **공인시험성적서(KOLAS 등) 기반 검증** 수행

### 2.2 (2) SHAP 기반 해석 강화 및 sLLM/Agent 기반 마련 — **진척도 55%**
- **계획서 기반 개발 내용**: xAI SHAP Value의 설명성 강화 및 sLLM/Agent를 통한 해석 결과 도출 기반 마련
- **수행 결과(무엇을 개발했는가)**  
  - SHAP 계산(TreeExplainer) 및 feature 중요도 \(mean(|SHAP|)\) Top-N 집계 자동화  
  - Top-N 결과를 CSV로 저장하고 막대그래프(PNG)로 시각화하는 산출물 생성  
  - 설명 입력으로 활용 가능한 구조화 산출물(JSON/샘플 입력) 생성  
- **증빙(부록 참조)**  
  - 파이프라인 및 산출물 목록: **부록 B-1**  
  - 네트워크/구성도(1차년도 단일노드 구조): **부록 B-2**  
- **미진행 및 보완(2차년도 이월)**  
  - sLLM/Agent 기반 “대화형 해석/설명” 구현(워크플로우 오케스트레이션, 툴 호출, 근거 제시)  
  - 설명 품질 평가 체계(정답셋/루브릭/Q&A Set) 구축 및 정량 평가

### 2.3 (3) LLM 기반 설명 인터페이스 시스템(Agent+RAG, SaaS 연계) — **진척도 30%**
- **계획서 기반 개발 내용**: LLM 기반 대화형 설명 인터페이스 및 Agent/RAG 연계 시스템 구축
- **수행 결과(무엇을 개발했는가)**  
  - 2차년도 운영화를 전제로 한 시스템 모듈 구성(클라이언트–API–예측–해석–DB–RAG/VectorDB) 설계  
  - 1차년도 로컬 파이프라인(단일 노드)과 2차년도 목표 아키텍처를 구분하여 구성도 정리  
- **증빙(부록 참조)**  
  - 목표 아키텍처 구성도(Agent/RAG/SaaS): **부록 B-2**  
- **미진행 및 보완(2차년도 이월)**  
  - 실제 UI/프로토타입, 플랫폼 API 연동, 운영 기능(권한/로그/모니터링/배포) 구현  
  - 응답시간 목표 및 운영 품질(보안 점검 포함) 검증

## 4. 2026년(2차년도) 개발 계획 및 일정(무엇을/얼마나/언제)
> 2차년도 목표: **1차년도 개발된 xAI 모델 최적화 및 경량화** + **LLM 기반 대화형 인터페이스·Agent 개발 및 SaaS 플랫폼 개발**

### 4.1 2026년 세부 추진 항목(계획서/2차년도 표 기반) 및 진척도 목표
> 2차년도(12개월) 세부내용은 “(1) xAI 모델 최적화 및 경량화”, “(2) 설명 인터페이스 시스템 확장 개발”, “(3) xAI 해석 증강용 파이프라인 구축”의 3개 축으로 구성한다.

#### (1) xAI 모델 최적화 및 경량화 — 연간 **100%**
- **(1-1) 성능 향상 최적화(40%)**  
  - 설명 모델의 처리속도/유형성 향상을 위한 모델·피처·학습전략 최적화
  - 성능/정확도 평가 체계 고도화(지표 정합화·평가셋 고정 포함)
- **(1-2) 모델 경량화/추론 최적화(40%)**  
  - 의사결정 트리 기반 단순화, 설명 모델 프로세스 개선, 모델 축소(Pruning/압축 등) 적용
  - 추론 시간/자원 사용량 목표 충족(서비스 적용 관점)
- **(1-3) 검증/인증(20%)**  
  - 운영 적용 전 성능 검증 및 공인시험성적서(KOLAS 등) 기반 인증 절차 수행

#### (2) 설명 인터페이스 시스템 확장 개발 — 연간 **100%**
- **(2-1) AI 결과 조회/분석 Agent 구현(40%)**  
  - AI 분석 결과/학습 이력 정보를 Agent 방식으로 획득·취합하여 설명 입력 데이터로 제공
- **(2-2) sLLM 기반 시각화·설명 인터페이스 개발(60%)**  
  - 분석/해석 결과 시각화 대시보드 및 대화형 설명 인터페이스(xUI) 구현
  - 사용자 피드백(학습 전략/피드백 형태)까지 확장 가능한 설명 템플릿/근거 제시 기능 포함

#### (3) xAI 해석 증강용 파이프라인 구축 — 연간 **100%**
- **(3-1) Agent Studio 기반 운영 기능 강화(30%)**  
  - Agent 데이터 조회/작업 및 관리(권한/모니터링 포함) 기능 강화
- **(3-2) 분석/해석 데이터 시각화 및 설명가능 AI 운영화(40%)**  
  - 분석 DB/해석 DB/Vector DB 기반 데이터 흐름 확립 및 시각화 모듈 개발
- **(3-3) SaaS 상용화 구조/운영 시스템 구축(30%)**  
  - 파이프라인 관리(Airflow 등), MLOps(MLflow 등), API Service, 로그/모니터링, 보안 점검 포함
  - 시범 운영 및 운영 프로세스(배포/장애대응/성능 모니터링) 구축
